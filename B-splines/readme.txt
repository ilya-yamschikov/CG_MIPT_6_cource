Базисные функции B-слайна.

Программа позволяет отобразить базисные функции B-сплайна.
Задаются параметры:
	Points count - количество точек сплайна (n)
	Spline degree - степень сплайна (k)
	Knots - узловой вектор (m)

По заданным параметрам выводятся n базисных фунций используемых для построения сплайна степени k на n точках
с использованием заданного вектора. 

Для запуска необходима установленная JVM версии 1.6 и новее.
